工程目录

|- _tests_  单元测试入口 
|- android  android 原生目录  
|- doc 文档操作手册相关  
|- ios ios原生目录入口  
|- index 工程入口
|- GlobalExpansionUtils  global拓展 日志，错误收集
|- src 源码目录  
|- src/asserts 共用图片目录  
|- src/components 共用组件存放目录  
|- src/contants 共用常量，环境，共用样式  
|- src/contants/Common.js  共用常量
|- src/contants/Config.js  配置
|- src/contants/Evn.js     环境
|- src/contants/Style.js   样式
|- src/models    dva model
|- src/pages     页面交互  
|- src/router    路由  
|- src/services  接口请求目录  
|- src/store     model注册
|- src/utils     基础工具类目录  
|- src/utils/Auth 
|- src/utils/FitSize    
|- src/utils/Help 
|- src/utils/IphoneXHelp 
|- src/utils/Network 
|- src/utils/Request 
|- src/utils/Toast 
|- src/utils/View
